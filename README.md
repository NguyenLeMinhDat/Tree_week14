## **Giới thiệu** 
Chương trình cho cây nhị phân theo yêu cầu BTVN.

## **Hướng dẫn chạy chương trình**
Giả dụ rằng bạn đã cài đặt trình biên dịch **gcc** hoặc bất kì trình biên dịch C nào khác.  
Sau khi clone repo về, hãy mở terminal lên, đảm bảo rằng bạn đang ở đúng vị trí thư mục, chạy câu lệnh sau để chạy chương trình:  

```
chmod +x ./run.sh
./run.sh
```
